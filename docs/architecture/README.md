# Architecture Diagrams

This folder contains editable Mermaid diagrams for SynosCD.

## Files

- `synoscd-context.mmd`: high-level system context and control flow.

## Editing Notes

- Keep diagrams source-first (`.mmd`) so they stay easy to review in pull requests.
- Prefer incremental updates as design decisions are made.
- Add new diagrams only when needed (e.g., reconcile flow, identity/RBAC boundaries).
